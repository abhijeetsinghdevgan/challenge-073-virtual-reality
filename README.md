# Aether VR - Virtual Reality Landing Page

## Challenge #073: Virtual Reality
A high-end, responsive landing page designed for a futuristic VR headset called **Aether**. The interface focuses on intuition, glassmorphic aesthetics, and immersive interactions.

### 🚀 Features
- **Intuitive Gesture Controls**: Design philosophy centered around small screen real estate and natural hand movements.
- **Glassmorphic UI**: Modern design using backdrop filters, subtle borders, and neon glows.
- **Dynamic Animations**: 3D parallax effects on hardware visuals and interactive feature cards.
- **Custom Interaction System**: A specialized cursor and follower system that reacts to interactive elements.
- **Responsive Design**: Seamless experience across mobile, tablet, and desktop devices.

### 🛠️ Tech Stack
- **Structure**: HTML5 Semantic Markup
- **Styling**: Vanilla CSS3 (Custom properties, Flexbox, Grid, Glassmorphism)
- **Logic**: Vanilla JavaScript (ES6+, DOM Manipulation, 3D transformations)
- **Icons**: FontAwesome 6
- **Typography**: Outfit (Google Fonts)

### 📸 Preview
The landing page features a hero section with a floating VR headset, feature cards with 3D hover effects, and a custom cursor system.
